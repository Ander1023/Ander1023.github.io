eclipse快捷键

tag：快捷键

Ctrl+D：删除当前行
Ctrl+1:快速修复
Ctrl+W 关闭当前Editer
Ctrl+M 最大化当前的Edit或View
Alt+/：自动完成、展开提示
Ctrl+/：自动注释当前行或者选择的多行
Ctrl+Shift+/：自动注释掉选择的代码块
Ctrl+Shift+F 格式化当前代码(注意输入法快捷键冲突)
Ctrl+Shift+o：自动引入包和删除无用包
Shift+Enter：在当前行的下方开始新行
Shift+Ctrl+Enter：在当前行插入空行(原理同上条)
Ctrl+Alt+↓ 复制当前行到下一行(复制增加)
Ctrl+Alt+↑ 复制当前行到上一行(复制增加)

idea快捷键

tag：快捷键

Ctrl+Y：删除当前行
Ctrl+D:复制当前行
Ctrl+G：定位到文件某一行
F2：定位至下一个错误处
Shift+F2：定位至前一个错误处
Alt+Insert：产生构造方法，get/set方法等
Alt+/：自动完成
Ctrl+/：使用//注释
Ctrl+Shift+/：使用/**/注释
Ctrl+Alt+L：格式化代码
Ctrl+Alt+I：自动缩进行
Ctrl+Alt+O：优化import
Ctrl+]：快速跳转至诸如{}围起来的代码块的结尾处
Ctrl+[：快速跳转至诸如{}围起来的代码块的开头处
Ctrl+W：选中整个单词
Shift+Enter：在当前行的下方开始新行
Ctrl+Alt+Enter：在当前行的上方插入新行
Alt+Shift+向下箭头：将行向下移动
Alt+Shift+向上箭头：将行向上移动
alt + shift 鼠标点 同时编辑多行

sublime快捷键

tag：快捷键

c ent 下行插入
c s ent 上行插入
c s k 删除整行
c k k 删除当前到行尾
c s d 复制到下一行
c / 当行注释
c s / 多行注释
t 向右缩进
s t 反向缩进
c s [ 折叠选中
c s ] 展开选中
c k 0 展开所有
c m 找括号
c j 多行代码合并为一行
c s ↑/↓ 移动一行代码
a f3 同名称编辑
c f 普通搜索框
c s f 文件夹搜索框
c p/g/r 文件/跳行/函数
c s p 命令框
F11 全屏模式
Shift+F11 免打扰模式

git命令

tag：命令

git init
git add file
git commit -m ""
git log
git reflog
git reset --hard id
git status
git diff
git checkout --file（拉取最近一次的提交》》》工作区）
git reset HEAD file(拉取最近一次的提交》》》暂存区【不修改本地】)
git rm file （删除一个文件【工作区与缓存区】）
git rm --cached file (删除一个文件【仅缓存区】)
git remote -v
git remote add origin url
git remote rm id
git push -u origin master 将本地master推送到origin主机
git pull （同步本地）
git fetch (同步版本库)
git branch branchname
git branch -a
git branch -d
git checkout branchname
git checkout -b branchname
git meger master
git stash 将本地修改进行存储

docker命令

tag：命令

docker run -it --name text -d centos:centos7 /bin/bash
(-it /bin/bash 临时运行一个，退出即删除，-d后台运行)
docker run id 运行一个容器
docker ps -a 查看当前的所有容器
docker search centos 查找
docker images 查看当前镜像
docker pull centos:centos7 下载镜像
